Now that you read how the module works, we are going to set up a basic test environment.

Pre-requisite

First, you need to have at least 2 Drupal instances

  • A master, which will push the synchronize order;
  • A slave, which will pull content and update it locally.

You can install both Drupal on the same physical machine, or over a network in which then can communicate.

Software installation

On server, enable this modules:

  • Yamm API
  • Yamm server

On client, enable this modules:

  • DataSync
  • Yamm API
  • Yamm client

You are ready to go, proceed to the next step.
